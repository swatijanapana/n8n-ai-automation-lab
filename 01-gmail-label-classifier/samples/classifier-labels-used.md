# 🏷️ Classifier Labels Used

This file lists the Gmail labels used in the Gmail Label Classifier workflow.

## Gmail Labels

| Label | Purpose |
|---|---|
| Bills & Utilities | Billing and utility-related emails |
| Alerts & Newsletters | General alerts, newsletters, and promotional emails |
| Shipping & Orders | Orders, purchases, packages, and delivery updates |
| Learning & Courses | Course, training, and learning platform emails |
| High Priority | Emails that may need urgent attention |
| Job Alerts | Job, recruiter, interview, and application-related emails |

## Notes

- These labels must exist in Gmail before the workflow applies them.
- The workflow assigns exactly one label to each email.
- Detailed classification logic is documented in `classifier-rules.md`.
