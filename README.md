## 📂 Projects

| Project | Description |

|--------|--------------------|

| 📧 `email-automation/01-gmail-label-classifier` | Classifies Gmail messages and applies labels using AI |  

| 📧 `email-automation/02-email-triage-assistant` | Summarizes emails, assigns priority, and suggests replies |
